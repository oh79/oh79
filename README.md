사람들이 겪는 문제를 기술로 해결하고, 실제로 사용되는 서비스를 만드는 데 집중하고 있습니다.

- 다양한 팀/공모전/대회 경험을 통해 “기술로 문제를 해결하는 방법”을 계속 검증하고 있습니다.
